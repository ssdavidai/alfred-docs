---
title: "List Secrets"
description: "List configured secret providers (values are masked)"
api: "GET /api/v1/openclaw/secrets"
---

Returns the output of `openclaw secrets` run inside the container. Secret values are masked by the OpenClaw CLI itself. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/secrets \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw secrets output (values are masked)"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "OPENROUTER_API_KEY=sk-or-v1-***\nHETZNER_S3_ACCESS_KEY=***"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw secrets` command output. Secret values are always masked by the CLI before being returned. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
