---
title: "List Skills"
description: "List all available OpenClaw agent skills"
api: "GET /api/v1/openclaw/skills"
---

Returns the output of `openclaw skills` run inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/skills \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw skills output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "vault-curator, vault-janitor, vault-distiller"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw skills` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
