---
title: "Restart Gateway"
description: "Restart the OpenClaw gateway container"
api: "POST /api/v1/openclaw/restart"
---

Restarts the OpenClaw gateway container via `docker compose restart openclaw`. All active connections will be terminated and devices will need to reconnect.

<RequestExample>

```bash cURL
curl -X POST https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/restart \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "message": "OpenClaw container restarted"
}
```

</ResponseExample>
