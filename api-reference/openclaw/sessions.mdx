---
title: "List Sessions"
description: "List all conversation sessions"
api: "GET /api/v1/openclaw/sessions"
---

Returns the output of `openclaw sessions` run inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/sessions \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw sessions output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "No active sessions"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw sessions` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
