---
title: "List Cron Jobs"
description: "List all scheduled cron jobs"
api: "GET /api/v1/openclaw/cron"
---

Returns the output of `openclaw cron` run inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/cron \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw cron output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "No cron jobs configured"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw cron` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
