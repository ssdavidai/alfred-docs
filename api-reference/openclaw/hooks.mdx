---
title: "List Hooks"
description: "List all registered event hooks"
api: "GET /api/v1/openclaw/hooks"
---

Returns the output of `openclaw hooks` run inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/hooks \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw hooks output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "No hooks registered"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw hooks` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
