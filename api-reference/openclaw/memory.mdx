---
title: "Search Agent Memory"
description: "Search over agent memory and conversation history"
api: "GET /api/v1/openclaw/memory"
---

Runs `openclaw memory [query]` inside the container. If a `query` parameter is provided, it is passed as an argument to the CLI command. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<ParamField query="query" type="string" optional>
  The search query text. Passed as a positional argument to `openclaw memory`. If omitted, lists all memory entries.
</ParamField>

<RequestExample>

```bash With Query
curl -X GET "https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/memory?query=vault%20organization" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash Without Query
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/memory \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw memory output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "No memory entries found"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw memory` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
