---
title: "List Agents"
description: "List all managed OpenClaw agents"
api: "GET /api/v1/openclaw/agents"
---

Returns the output of `openclaw agents` run inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/agents \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw agents output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "No agents configured"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw agents` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
