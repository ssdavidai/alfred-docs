---
title: "Run Diagnostics"
description: "Run health checks and auto-fixes for OpenClaw gateway"
api: "POST /api/v1/openclaw/doctor"
---

Runs `openclaw doctor` inside the container. This performs comprehensive health checks and attempts auto-fixes. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X POST https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/doctor \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw doctor output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "All checks passed"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw doctor` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
