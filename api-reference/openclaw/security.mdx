---
title: "Security Audit"
description: "Audit OpenClaw gateway security configuration"
api: "GET /api/v1/openclaw/security"
---

Runs `openclaw security` inside the container. This performs a security audit and returns findings. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/security \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw security output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "Security audit complete: no issues found"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw security` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
