---
title: "Gateway Health"
description: "Check OpenClaw gateway health status"
api: "GET /api/v1/openclaw/health"
---

Returns the health status of the OpenClaw gateway by running `openclaw gateway health` inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/health \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw gateway health output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "Gateway is healthy"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw gateway health` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
