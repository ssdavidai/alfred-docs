---
title: "Gateway Status"
description: "Get detailed OpenClaw gateway status information"
api: "GET /api/v1/openclaw/status"
---

Returns detailed status information about the OpenClaw gateway by running `openclaw gateway status` inside the container. If the CLI outputs valid JSON, it is returned parsed. Otherwise, the raw text is returned in a `raw` field.

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/openclaw/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Structured Response (if openclaw outputs JSON)
{
  "...": "Structure depends on openclaw gateway status output"
}
```

```json Raw Response (if openclaw outputs text)
{
  "raw": "Gateway status: running"
}
```

</ResponseExample>

<Note>
  The response format depends on the `openclaw gateway status` command output. If the command outputs valid JSON, it is returned as-is. Otherwise, the raw text output is returned in a `raw` field.
</Note>
