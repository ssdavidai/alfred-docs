---
title: "Trigger Manual Backup"
description: "Start a manual backup of the Alfred instance"
api: "POST /api/v1/admin/backups/trigger"
---

Trigger a manual backup using restic. This is a fire-and-forget operation that returns immediately with 202 Accepted. The backup runs in the background.

<RequestExample>

```bash curl
curl -X POST -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/admin/backups/trigger
```

</RequestExample>

<ResponseExample>

```json 202
{
  "message": "Backup triggered"
}
```

</ResponseExample>

<Note>
  The backup runs asynchronously with a 600-second timeout. The endpoint returns 202 immediately without waiting for the backup to complete. Errors are logged server-side. Use the list snapshots endpoint to verify backup completion.
</Note>
