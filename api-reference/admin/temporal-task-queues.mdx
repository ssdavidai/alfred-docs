---
title: "Describe Temporal Task Queue"
description: "Get information about a specific Temporal task queue"
api: "GET /api/v1/admin/temporal/task-queues/{name}"
---

Retrieve information about a Temporal task queue. Runs `temporal task-queue describe --task-queue <name> --output json` inside the temporal container and returns the parsed output.

## Path Parameters

<ParamField path="name" type="string" required>
  The name of the task queue to describe.
</ParamField>

<RequestExample>

```bash curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/admin/temporal/task-queues/my-task-queue
```

</RequestExample>

<ResponseExample>

```json
{
  "taskQueueName": "my-task-queue",
  "taskQueueType": "Workflow",
  "pollers": [
    {
      "identity": "worker@alfred-acme-worker-1",
      "lastAccessTime": "2026-02-26T10:15:30Z",
      "ratePerSecond": 100
    }
  ],
  "taskQueueStatus": {
    "backlogCountHint": 0,
    "approximateBacklogAge": "0s",
    "tasksAddRate": 2.5,
    "tasksDispatchRate": 2.5
  },
  "taskIdBlock": {
    "startId": 1000,
    "endId": 2000
  }
}
```

</ResponseExample>

<Note>
  The response structure depends on the Temporal CLI's output. If it outputs non-JSON text, the response will be `{ "raw": "..." }` instead.
</Note>
