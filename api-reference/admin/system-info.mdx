---
title: "Get System Information"
description: "Retrieve system uptime, disk, and memory information"
api: "GET /api/v1/admin/system/info"
---

Get basic system information including uptime, disk usage, and memory statistics. Each field contains raw text output from the respective system command (`uptime -p`, `df -h /mnt/encrypted`, `free -m`). If any command fails, the corresponding field will contain `"unknown"`.

<RequestExample>

```bash curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://alfred-acme.tailnet.ts.net:3100/api/v1/admin/system/info
```

</RequestExample>

<ResponseExample>

```json
{
  "uptime": "up 5 days, 3 hours, 42 minutes",
  "disk": "Filesystem      Size  Used Avail Use% Mounted on\n/dev/mapper/encrypted  200G   85G  115G  43% /mnt/encrypted",
  "memory": "              total        used        free      shared  buff/cache   available\nMem:           7982        4915        1228         256        1838        2810\nSwap:          2047         512        1535"
}
```

</ResponseExample>

<Note>
  All three fields contain raw text output from the respective system commands. If any command fails, its field will contain `"unknown"`. The disk output shows usage for `/mnt/encrypted`, the encrypted data volume containing all Alfred data.
</Note>
