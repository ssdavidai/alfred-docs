---
title: "Get Worker Status"
description: "Retrieve the overall status of all Alfred daemon workers"
api: "GET /api/v1/workers/status"
---

Returns the current operational status of all Alfred workers (curator, janitor, distiller, surveyor). This endpoint runs `alfred status` inside the Alfred container and returns the raw text output.

<Note>
  This endpoint always returns raw text output from `alfred status`, not structured JSON. The response contains a single `raw` field with the status text.
</Note>

<RequestExample>

```bash cURL
curl -X GET https://alfred-acme.tailnet.ts.net:3100/api/v1/workers/status \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json Response
{
  "raw": "curator: running\njanitor: running\ndistiller: stopped\nsurveyor: not configured"
}
```

</ResponseExample>

<ResponseField name="raw" type="string">
  Raw text output from `alfred status`. Contains per-worker status information as human-readable text.
</ResponseField>
